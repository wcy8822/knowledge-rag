## 核心功能

### 1. 本地文档扫描

- 递归扫描指定目录
- 识别 MD/SQL/代码文件
- 自动业务分类
- 生成 CSV 清单

### 2. 分批次向量化

- 支持 MD/SQL/代码文件
- 分批次处理（≤500 文件/批）
- 内存峰值 ≤12GB
- 断点续传

### 3. 自动化运维

- 文件监控（watchdog）
- 自动更新向量
- 定时校验一致性
- 本地日志记录

### 4. 本地通用接口

- REST API（127.0.0.1）
- 支持多种检索方式
- 数据脱敏
- 多 AI 调用支持
