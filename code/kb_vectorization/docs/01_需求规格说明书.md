## 3. 功能需求

### 3.1 需求 FR-01: 本地文档扫描 & 统计

| 编号 | 名称 | 优先级 |
|------|------|--------|
| FR-01 | 本地文档扫描与统计 | P0 |

#### 3.1.1 功能描述

递归扫描指定目录，识别所有可向量化文件，并生成统计清单。

#### 3.1.2 功能详情

| 子功能 | 描述 |
|--------|------|
| 递归扫描 | 支持深度扫描指定目录及子目录 |
| 文件类型识别 | 识别 MD、SQL、代码文件（支持扩展配置） |
| 业务分类 | 自动按目录/内容特征分类 |
| 统计输出 | 生成 CSV 格式清单 |

#### 3.1.3 业务分类规则

| 分类 | 识别规则 |
|------|---------|
| 资讯 | 目录包含"资讯"、"news"、"report" |
| 商户画像 | 目录包含"商户"、"customer"、"profile" |
| 技术配置 | 目录包含"config"、"setting"、"配置" |
| 项目汇报 | 目录包含"汇报"、"report"、"总结" |
| 数据表映射 | 目录包含"mapping"、"table"、"schema" |
| SQL 调试 | 目录包含"sql"、"debug"、"query" |

#### 3.1.4 输出格式（CSV）

```csv
路径,类型,分类,大小(字节),修改时间,是否有效
/path/to/file1.md,MD,商户画像,2048,2026-03-01 10:30:00,true
/path/to/file2.sql,SQL,数据表映射,4096,2026-02-28 15:20:00,true
```

#### 3.1.5 技术要求

- 支持配置多个扫描目录
- 支持排除目录（如 `.git`, `node_modules`）
- 支持增量扫描（只检查变化的文件）
- 扫描结果本地存储不上传

#### 3.2.2 批次规则

| 代码 | 按函数/类分块 | 代码语义向量化 |
| MySQL 表结构 | 从 SQL 文件提取 | 表结构向量化 |

#### 3.2.4 MD 文件预处理规则

| 规则 | 说明 |

| 2 | 合并连续空行为单行 |
| 3 | 保留段落语义，按 `\n\n` 分块 |
| 4 | 移除代码块语法高亮标记 |
| 5 | 移除 HTML 标签 |
| 6 | 处理表格为可读文本 |
| 7 | 保留特殊业务词汇（如 SKU、GMV） |
