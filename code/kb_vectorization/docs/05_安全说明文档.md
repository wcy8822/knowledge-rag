### 3.2 数据脱敏

#### 脱敏模式

| 模式 | 说明 | 适用场景 |
|------|------|---------|
| strict | 严格模式，脱敏所有敏感信息 | 高安全要求 |
| moderate | 中等模式，只脱敏明显敏感信息 | 默认推荐 |
| none | 不脱敏 | 仅在完全信任环境 |

#### 脱敏规则

```yaml
security:
  data_sanitization:
    enabled: true
    mode: "moderate"
    patterns:
      - name: "手机号"
        pattern: "\\d{11}"
      - name: "银行卡号"
        pattern: "\\d{15,19}"
      - name: "身份证号"
        pattern: "\\d{17}[\\dXx]"
      - name: "邮箱"
        pattern: "\\b[A-Za-z0-9._%+-]+@[A-Za-z0-9.-]+\\.[A-Z|a-z]{2,}\\b"
```
