## 5. 常见问题

### 5.1 安装问题

#### Q: Python 版本不满足要求怎么办？

A: 使用 pyenv 安装指定版本的 Python：

```bash
# 安装 pyenv
brew install pyenv

# 安装 Python 3.10+
pyenv install 3.10.0
pyenv global 3.10.0
```

#### Q: pip 安装依赖失败？

A: 使用国内镜像源：

```bash
pip3 install -r requirements.txt -i https://pypi.tuna.tsinghua.edu.cn/simple
```

### 5.2 运行问题

#### Q: 内存使用过高怎么办？

A: 减小批次大小：

```yaml
# 编辑 config/config.yaml
vectorize:
  batch_size: 100  # 从 500 减小到 100
```

#### Q: 向量化速度慢？

A: 检查以下几点：

1. 文件是否过大（> 10MB）
2. 是否使用了 SSD
3. CPU 占用是否正常

#### Q: API 无法访问？

A: 检查：

```bash
# 检查服务是否运行
./scripts/check_status.sh

# 检查端口占用
lsof -i :8000

# 查看日志
tail -f logs/api.log
```

### 5.3 搜索问题

#### Q: 搜索结果不准确？

A: 尝试：

1. 使用混合检索（默认）
2. 增加返回结果数（top_k）
3. 检查查询关键词是否准确

#### Q: 搜索没有结果？

A: 检查：

1. 是否已完成向量化
2. 查询关键词是否正确
3. 文件分类是否正确

### 6.2 常见错误处理

#### 错误：ModuleNotFoundError

```bash
# 解决：重新安装依赖
pip3 install -r requirements.txt
```

#### 错误：Permission denied

```bash
# 解决：设置执行权限
chmod +x scripts/*.sh
```

#### 错误：MemoryError

```bash
# 解决：减小批次大小
./scripts/vectorize.sh --batch-size 50
```

#### 错误：Connection refused

```bash
# 解决：检查服务状态
./scripts/check_status.sh

# 重启服务
./scripts/stop_api.sh
./scripts/start_api.sh --daemon
```
